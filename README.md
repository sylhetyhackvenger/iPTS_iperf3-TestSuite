# -iPTS-iperf3-TestSuite
iPerf3 Test Suite v2.0 is a powerful network performance analysis tool featuring 71 pre‑built iPerf3 commands across 7 categories (Bandwidth, TCP, UDP, Wi‑Fi, Latency, Infrastructure, Advanced). For authorised network troubleshooting, ISP verification, VPN testing, and capacity planning. 🚀
